插件

在打开ros2工程前需要安装几个插件

  • CMake

  • CMake Tools

  • Python

  • Ros

  • XML Tools

配置

c_cpp_properties.json

这个文件用于配置编译条件，我这里用的是foxy版本的ros2，其他版本可以参考着改

launch.json

这个文件用于启动ros2 package，target选项用于配置package运行脚本，一般用相对位置比较合适

settings.json

vscode的基本配置，关联一些cpp的库、其它文件类型

tasks.json

编译任务配置，debug需求的可以改成debug编译类型

调试
